# Features

Detailed specifications for Laudasist features.

## Feature Index

| Feature | Status | Description |
|---------|--------|-------------|
| [Worship Session](./worship-session.md) | Phase 2 | Local-only quick worship leading |
| [Live Service](./live-service.md) | Phase 5 | Full multi-viewport broadcasting |
| *Song Editor* | Planned | Future detailed spec |

## Quick Reference

### Worship Session (Fast Service)
- 2-column layout: Song Search | Live View
- No preview step — song goes live immediately
- Local view only, no broadcasting
- Chord display, transpose, style options

### Live Service
- 4-column presenter dashboard
- Preview → Live workflow
- Multi-viewport broadcasting
- Concurrent presenter support
